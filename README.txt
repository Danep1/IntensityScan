Программа для анализа яркости пикселей изображения вдоль прямой.
0) Програмные требования (requirements.txt):
    Python 3.8+
    OpenCV
    matplotlib
    numpy
0.1) Можно быстро установить модули командой в командной строке:
    pip install -r requirements.txt

Как пользоваться:
1) Запуск программы осуществляется через консоль:
    python main.py image_name.txt
1.1) "image_name.txt" - относительный путь от main.py
1.2) Изображение должно быть квадратным!!!
1.3) Рекомендую обработать изображение перед анализом: обрезать (до квадрата), повысить контрастность и яркость.

2) При запуске вы увидите выше исходное изображения, но замыленое (так и надо)

3) На изображение нужно нанести масштаб (нужен отрезок на изображении длинной 1 см, например линейка):
    Нажмите правой кнопкой мыши на начало ориентира и удерживайте, протащите мышку до конца, а затем отпустите.
    На экране появится серый отрезок, соотвествующий выбранному масштабу
3.1) В случае ошибки при нанесении масштаба можно нанести отрезок заново, повторив п.3

4) Далее нужно выбрать отрезок, вдоль которой будет вычисляться яркость.
    Нажмите левой кнопкой мышки на начало отрезка и удерживайте, протащите мышку до конца линии, а затем отпустите.
    На экране появится график зависимости яркости пикселя от расстояния в см, согласно выбранному масштабу
4.1) В случае ошибки при выборе линии её можно нанести заново, повторив п.4

5) В отдельном окне высветиться соотвествующий график. Сохранить его можно, нажав соотвествующую кнопку в этом окне.

Фичи:
- Нажатие средней кнопки мыши выводит в консоль координаты текущего пикселя и его яркость.
- Однократное нажатие левой кнопки мыши выводит в консоль координаты текущего пикселя и усредненную яркость области 3 на 3.
    Именно это усредненное значение откладывается на графике
